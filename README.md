# Запуск проекта

`git clone <https_key>`

```
1. Создание виртуального окружения
    `python3 -m venv <venv_name>`
2. Активация виртупльного окружения
    `source venv/bin/activate`
3. Установка библиотек
    `pip install -r requirements.txt`
```
